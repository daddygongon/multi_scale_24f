1. 課題

1.1. directoryの整理

  • chunkingの基本は名前をつけることです．
  • 一眼見ただけで，内容がわかる名前にしましょう．
  • w1とかでは中身を忘れてしまいます．
  • tree -d で中身まで予測できるようにしましょう．
> tree -d -I 'images' #for w4_hyper_card
.
├── bat_fzf
├── emacs_browser_url
├── github_private_invite
├── hyper_card
│   ├── chunking
│   └── semi_lattice
├── tldr
└── tree
