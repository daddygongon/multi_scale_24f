* 運指(Fingering)
| emacs   | Ctrl-a           | pinky -> ctrl | ring -> a      |               |
| windows | Atl-v            | thumb -> alt  | index -> v     |               |
| Mac     | com-v            | thumb -> com  | index -> v     |               |
|         | command or Apple |               |                |               |
|         | com-x com-v      | thumb -> com  | index -> x     | index -> v    |
|         | opt-shift-i      | thumb -> opt  | middle-> shift | r-middle -> i |
