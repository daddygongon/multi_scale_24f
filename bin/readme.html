1 ~/bin

userが自分で使うcommandの置き場が~/binです． ここにあるいくつかのcommandをそこに置いて使ってください．

2 パスを通す

2.1 tldr(too long don't read)

fishで使えるようにパス(path)を通す必要があります．

  • emacs ~/.config/fish/config.fish
set PATH ~/bin $PATH

とします．

2.2 $PATH

実行可能コマンドの置き場所を パソコンに教えてあげる必要があります． そうでないと，パソコンは全ディレクトリを探す ことになります． 現在のshell環境でどこに通っているかは，

  • echo $PATH$

で確認できます．

config.fishを変更したら，それをfishに知らせる必要あります．

  • shellを新たに起動するか，
  • source ~/config/fish/config.fish

でできるはず．echoして確かめてください．

2.3 その他の置き場

systemが使うのや，そのほかのがあります．

  • /bin, /sbin,
  • /usr/bin
  • /usr/local/bin

apt(brew), gemなどでバリエーションがありますので， commandのinstallマニュアルを参照ください．
