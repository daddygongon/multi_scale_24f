4 hyper_card

my_helpのinstall
  1. gem install my_help
    1. FilePermissionErrorがでたら， sudo を付ける．
  2. my_help で動作確認
  3. my_help init でdefault files, editorなどを調整
  4. my_help add_defaults が必要かも．．
  5. my_help -l でdefault helpsを確認．
