1. お題:Fibonacci数列

recursion(再帰)です． 

fib(0) = 0
fib(1) = 1
fib(n) = fib(n-1)+fib(n-2) for n>=2
0 1 1 2 3 5 8 13 21 ...
index n 0 1 2 3
val fin(n) 0 1 1 2
