1. clip.exe

unixのコマンド出力をそのままPaste baord(mac)， クリップボード(win),x-selection(unix)へ保存して， pasteすることができます．

1.1. wsl

  • ls -1 | clip.exe

1.2. mac

  • ls -1 | pbcopy

1.3. linux

Xを介しての場合は，以下の通り． $ sudo apt-get install xsel # ubuntu or $ brew install xsel # cent $ cat hoge.txt | xsel –clipboard –input # on directories $ alias pbcopy='xsel –clipboard –input' # on ~/.config/fish/config.fish 誰か，Xがない場合でもできる方法を知ってたら教えてください．
